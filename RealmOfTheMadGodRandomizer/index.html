About This Tool

Realm of the Mad God is a massively multiplayer online bullet hell shooter that features a diverse roster of character classes, each with unique abilities and playstyles. With so many options available, it can be challenging to decide which class to play next.

This randomizer helps you break out of your comfort zone by randomly selecting a class for you to play. Use the sidebar to filter which classes you want to include in the randomization—perfect for when you want to try something new but still have some control over your options.

How to Use the Randomizer

Simply click the "Generate Class" button to randomly select a class from your enabled options. You can use the checkboxes in the sidebar to enable or disable specific classes. Use the "Select All" and "Deselect All" buttons for quick filtering. The re-roll button (↻) lets you generate a new class without starting from scratch.
