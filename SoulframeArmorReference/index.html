About This Tool

Welcome to the Soulframe Armor Reference, your comprehensive database for all armor in Soulframe! This tool provides a sortable and searchable table of all available armor pieces with their complete statistics and attributes.

Use the search bar to find specific armor by name, or utilize the filter dropdowns to narrow down armor by slot, set, virtue requirement, or rarity. Click on any column header to sort the table by that attribute.

How to Use the Reference

Click on any armor name to view its full description and detailed statistics including defence values, attunements, and virtue requirements. The table displays key statistics for quick comparison between armor pieces.

The filters can be combined to find exactly the armor type you're looking for - for example, all Rare helms that require Courage, or all armor pieces in the Mockery set.
